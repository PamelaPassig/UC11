Projeto: Leiloes

Sobre: O projeto Leilao é um sistema desktop desenvolvido em Java com o objetivo de gerenciar um leilão de produtos. A aplicação possui três telas principais: Cadastro de Produtos, Listagem de Produtos e Detalhes dos Produtos. Na tela de cadastro, o usuário pode inserir informações como nome, descrição, valor inicial e data do leilão. A listagem exibe todos os produtos cadastrados em forma de tabela, permitindo uma visualização rápida e organizada. A tela de produtos permite visualizar os detalhes completos e editar ou excluir itens.

Tecnologia usada: O projeto utiliza Java como linguagem principal, com desenvolvimento na IDE NetBeans, e SQL para persistência dos dados, por meio de um banco de dados relacional como MySQL. A conexão entre Java e o banco é feita com JDBC (Java Database Connectivity), garantindo que os dados sejam armazenados e consultados com segurança e eficiência.
